## 项目简介
场磨式传感器输出正弦波信号，幅值与电场强度成正比。本项目通过 FFT+ 时域特征 + 机器学习/深度学习的方法来预测直流电压，并判断压板投/退状态。

**主要特点：**
- 完整的数据处理流水线（数据库导出 → 数据清洗 → 模型训练 → 评估）
- 多种传统 ML 模型（线性、RandomForest、XGBoost、LightGBM、CatBoost 等）
- 多种深度学习模型（LeNet、LeNet-Hybrid、LeNet-BiPath）
- 完整的评估指标（MAE、RMSE、百分位误差、符号准确率、投/退判断准确率等）
- 分电压段详细评估，方便调试和问题定位

## 原理

场磨式传感器输出正弦波信号，幅值与电场强度成正比：

| 压板状态 | 下端子电压 | 传感器输出 |
|---------|-----------|-----------|
| 退（断开） | 0 V | 幅值小（≈0） |
| 投（接通） | -110 V DC | 幅值大 |

对 512 点波形做 FFT 提取基波幅值，结合温湿度等环境参数和时域统计特征，通过模型预测电压，与阈值比较判别。

## 输入参数一览

### 传感器硬件参数
| 参数 | 值 | 说明 |
|------|-----|------|
| 采样率 | 15873 Hz | 场磨传感器 ADC 采样频率 |
| 每帧点数 | 512 点 | 单次采集的波形长度 |
| 转子频率 | 213.8 Hz | 场磨旋转叶片调制频率 |
| 测量范围 | -110 V ~ +110 V | 直流电压 |
| 数据类型 | 电压值 (V) | 已标定，非 ADC 原始值 |

### 模型输入特征（共 16 维）

| # | 特征 | 来源 | 物理含义 | 与电压相关性 |
|---|------|------|---------|-------------|
| 1~10 | A1 ~ A10 | 512 点波形 → FFT 前 10 次谐波幅值 | 各次谐波分量大小，A1（基波）为转子频率 213.8Hz 处的幅值，与电场强度成正比 | **A1 为主信号**，A2~A10 反映传感器机械状态 |
| 11 | temp | 传感器内置温度计 | 环境温度，影响放大器增益和 ADC 参考电压 | 弱相关（温度补偿） |
| 12 | humid | 传感器内置湿度计 | 环境湿度，影响绝缘电阻和漏电流 | 弱相关（湿度补偿） |
| 13 | rpm | 转子转速测量 | 场磨电机转速，影响感应电流幅值 | 中等相关（转速补偿） |
| 14 | Vpp | 波形峰峰值（`Vpp = max(ac) - min(ac)`） | 信号最大波动范围，反映总能量 | 与 A1 相关，但携带互补信息 |
| 15 | Kurtosis | 峭度（Fisher=False） | 波形尖锐度，正态分布 = 3 | 反映传感器异常或噪声水平 |
| 16 | Skewness | 偏度 | 波形不对称度，正 = 正半周更大 | 反映机械偏心或叶片磨损 |

**特征推导流程:**
```
512 点原始波形 (512,)
    │
    ├── 去直流：ac = wave - mean(wave)
    │
    ├── FFT ─── 前 10 次谐波幅值 ─── A1 ~ A10（频域特征）
    │
    ├── 时域统计 ─── Vpp, Kurtosis, Skewness（load_jsonl 时预提取）
    │
    └── 传感器 ─── temp（温度）, humid（湿度）, rpm（转速）
```

**特征使用说明:**
- **传统线性模型（linear_model）**：使用全部 16 维特征
- **树模型（RandomForest, ExtraTrees, XGBoost, LightGBM, CatBoost）**：使用全部 16 维特征
- **LeNet-Hybrid / LeNet-BiPath**：使用三路独立输入 — 原始波形(512点) + FFT谐波(10维) + 环境/时域参数(6维)

**结论：**
- **RandomForest**：MAE **1.41 V**（传统 ML 最佳），±5% 精度 **89.84%**
- **LightGBM**：MAE 2.51 V，投/退判断最优（90.33%），且模型轻量
- **CatBoost**：MAE 2.41 V，与 LightGBM 精度相当
- **XGBoost / ExtraTrees**：MAE 3~5 V，表现中等
- **SVR / 二次多项式 / 线性**：误差较大，仅作为基线

### 各算法适用场景
| 场景 | 推荐算法 | 原因 |
|------|---------|------|
| **快速验证基线** | `linear_model` / `quadratic_model` | 理解数据基础规律 |
| **传统 ML 最佳精度** | `random_forest_model` | MAE 1.41 V，±5% 精度 89.84% |
| **生产部署（轻量、速度快）** | `lightgbm_model` | MAE 2.51 V，模型小（约 10KB） |
| **生产部署（精度均衡）** | `catboost_model` | MAE 2.41 V，开箱即用 |
| **端到端（无需 FFT 提取）** | `lenet` | 原始波形直出，适合嵌入式 |
| **最高精度（有 GPU）** | `lenet_hybrid` | 三路融合 + 时域特征 |
| **解决正负号错误** | `lenet_bipath` | 双路径分别预测大小/符号 |

## 安装与使用

### 环境准备
```bash
cd swa
uv sync
```

### 运行模式
数据源由 `src/swa/config/settings.py` 中的 `data_source.mode` 控制：

```python
data_source.mode = "db"     # 从达梦数据库读取（在公司用）
data_source.mode = "local"  # 从本地 JSONL 文件读取（回家离线用）
```

```bash
# 按配置模式运行（默认最近 20 条）
uv run python main.py

# 指定条数
uv run python main.py --limit 100

# 按从机地址和时间筛选
uv run python main.py --slave-id 1 --since "2026-06-01"

# 完整分析 + 生成 2×2 图表
uv run python main.py --plot
```

### 训练模型

#### 传统 ML 模型
```bash
# 训练单个模型
uv run python scripts/train_model.py --algorithm linear_model
uv run python scripts/train_model.py --algorithm random_forest_model
uv run python scripts/train_model.py --algorithm xgboost_model

# 训练所有模型（推荐！数据只加载一次，速度快）
uv run python scripts/train_model.py --all
```

#### 深度学习模型
```bash
uv run python scripts/train_dl.py --algorithm lenet
uv run python scripts/train_dl.py --algorithm lenet_hybrid
uv run python scripts/train_dl.py --algorithm lenet_bipath
```

不指定 `--algorithm` 时，默认使用 `settings.py` 中配置的算法。

### 预测与状态判别

```bash
uv run python main.py --limit 20
```

输出格式：
```
  110V  slave=1  2026-06-10 10:58:39  电压= +99.94V  投
   -50V  slave=1  2026-06-10 17:24:16  电压= -48.23V  投
    10V  slave=1  2026-06-11 09:15:33  电压=  +8.12V  退
```

### 投/退判断逻辑

```
预测电压的绝对值 > threshold_abs → "投"（有电场，压板接通）
预测电压的绝对值 < threshold_abs → "退"（无电场，压板断开）
```

阈值 `threshold_abs` 在 `settings.py` 中设置，默认 30V。
